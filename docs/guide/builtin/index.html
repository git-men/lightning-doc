内置功能

安装完lightning，登录进入lightning的管理页面，我们可以看到它内置并开启了以下功能：

导航菜单功能#

以下功能可以通过导航栏菜单访问：

  • 权限管理，用于管理系统的账号与角色，基于django的auth应用。
    • 账号管理，管理系统用户账号。
    • 角色管理，管理系统角色，并可以赋予角色不同的权限。
  • 开发管理，用于在本系统内进行可视化定制开发，包括导航栏菜单、页面配置等。
    • 菜单管理，用于管理系统导航栏菜单。
    • 页面配置，用于管理数据模型对应的页面配置。

这些功能的菜单项可以在菜单管理里面订制其标題、图标、权限等。

系统设置功能#

在系统界面右上角，光标悬停在登录名上面时，可以看到系统设置菜单项: 点击进入设置面板，在此面板，可以配置以下选项：

  • 网站设置，可以修改网站的名称和Logo。
  • 短信配置，使用腾讯云或阿里云的短信通知服务进行验证码发送或信息通知等功能。
  • 邮件配置，配置系统通知中的邮件信息。
  • 上传配置，配置文件上传使用的存储后端，可以选本地存储、阿里云OSS以及腾讯云COS。

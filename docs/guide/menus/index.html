导航菜单配置

lightning允许我们在系统运行时实时调整菜单的结构和排序，并实现以对不同角色可见菜单的控制。

点左系统导航栏菜单的 开发管理->菜单管理可进入菜单管理界面。在此，可以查看系统目前的菜单以及可以对其进行调整，包括增加、分组、修改、删除等。

菜单列表页#

我们可以通过拖拽的方式对菜单进行排序、分组等。

创建菜单页#

通过此表单，我们可以创建菜单组或菜单项，当创建菜单项时，需要指定菜单项页面类型，来创建列表页还是表单页等，每个菜单组或菜单项均可指定可以访问该菜单的角色。
